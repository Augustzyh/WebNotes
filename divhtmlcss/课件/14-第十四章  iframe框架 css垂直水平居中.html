一、iframe内联框架

1、iframe的属性

  • width 定义iframe的宽度；
  • height 定义iframe的高度；
  • frameborder 规定是否显示框架周围的边框
    • 0无
    • 1有
  • src 规定在框架中显示的文档的 URL。
  • scrolling 规定是否在框架中显示滚动条。
    • yes 显示滚动条
    • no 无
    • auto 根据内容自动显示滚动条
  • name 规定框架的名称

2、您可以把需要的文本放置在 <iframe>和 </iframe> 之间，这样就可以应对无法理解 iframe 的浏览器。

四、CSS实现垂直居中水平居中

参考文献：http://blog.csdn.net/freshlover/article/details/11579669

1、Absolute margin

1.{  

2.  margin: auto;  

3.  position: absolute;  

4.  top: 0; left: 0; bottom: 0; right: 0;  

5.}

缺点：
1.必须声明高度

2、浏览器窗口居中

1.{

2.    margin: auto;

3.    position: fixed;

4.    top: 0;

5.    left:0;

6.    bottom: 0;

7.    right:0;

8. }

3、边栏居中

如果你要设置一个固顶的头或增加其他的边栏，只需要在内容块的样式中加入像这样的CSS样式代码：top:70px;bottom:auto;由于已经声明了margin:auto;，该内容块将会垂直居中于你通过top,left,bottom和right属性定义的边界框内。
你可以将内容块固定与屏幕的左侧或右侧，并且保持内容块垂直居中。

使用left:0;right:auto;固定与屏幕左侧

1.{

2.

3.   margin: auto;

4.    position: absolute;

5.    top:0;

6.    left:20px;

7.    bottom: 20px;

8.    right:auto;

9.}

使用right:0;left:auto;固定于屏幕右侧

1.{

2.

3.   margin: auto;

4.    position: absolute;

5.    top:auto;

6.    left:20px;

7.    bottom: 20px;

8.    right:0;

9.}

修改给top:auto也能定义顶部，bottom:auto定义底部

1.{

2.    margin: auto;

3.    position: fixed;

4.    top:auto;

5.    left:auto;

6.    bottom: 20px;

7.    right:30px;

8. }
